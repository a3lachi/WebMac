# webmac
Check webmac.vercel.app or clone repo, and run : npm install ; npm run dev ; 

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
